# Brainstorm · 시 짓기 보드

> 무인 세션 · AI 자체 판단. 토픽 명세 + 실제 교실 사용 흐름 + YAGNI 원칙 기반.

## 사용자 인텐트 (가정)
- **작성자**: 학생 1인 1기기. 머리글자 단어를 받고 줄별로 한 문장씩 시 작성.
- **저장자**: 같은 학생이 자신의 작품을 모아서 학기 동안 갤러리로 관리.
- **발표자**: 선생님이 TV에서 무작위 한 작품을 띄워 발표 진행.
- **출판자**: 선생님 또는 학생 본인이 학기말 PDF 시집을 만들어 출력·배포.

## 꼭 필요한 기능 (MUST · 5개)
1. **머리글자 입력 → 줄 자동 생성**: 단어 한 칸 입력 후 `시작` 누르면 글자 수만큼 줄 입력 칸이 펼쳐진다. 한글뿐 아니라 영어·숫자도 머리글자로 허용 (단, 화이트스페이스는 무시).
2. **각 줄에 글자/음절 카운터**: 현재 줄의 글자 수와 단어 시작 글자 강조. 한국어 음절 단위 카운트가 핵심.
3. **갤러리 저장 (localStorage 1차)**: `저장` 누르면 시 작품이 갤러리에 카드로 누적. 새로고침 후에도 유지. 작품 삭제 가능.
4. **발표 모드 룰렛**: 갤러리 작품 중 1개를 무작위로 뽑아 풀스크린 카드로 표시. 키보드 좌·우로 이전/다음, ESC로 종료.
5. **학기말 시집 PDF**: 갤러리의 모든 작품을 1) 표지 → 2) 차례 → 3) 작품 N개 → 4) 발문 순서로 묶어 단일 PDF로 다운로드. 학급명·학기 제목을 표지에 노출.

## 있으면 좋은 기능 (SHOULD · 3개)
1. **단일 작품 카드 PNG 내보내기**: html2canvas로 카드 이미지 저장 → 학생이 부모님 단톡방 등에 자유롭게 공유 가능 (자기 자신만).
2. **File System Access API 폴더 미러**: 지원 브라우저(Chromium 계열)에서 폴더 한번 지정하면 작품 추가/수정 시 자동으로 JSON 백업 파일 생성. 미지원 브라우저(Firefox·Safari)는 조용히 비활성.
3. **JSON Import/Export**: 갤러리 전체를 json 파일로 백업·복원. 학년 말 다음 학기로 이관, 또는 친구에게 자기 작품 묶음 전달용.

## 절대 금지 기능 (MUST NOT · 4개)
1. **AI 자동 작시 / 추천 문장**: 토픽 의도와 정면 충돌. "내가 직접 쓰는" 활동.
2. **서버 업로드 / 공유 링크**: 학생 창작물 외부 전송 금지. 모든 저장은 클라이언트 측 (localStorage / FS Access / 다운로드 PDF).
3. **검열·필터링 모델 / 욕설 자동 차단**: 교사 재량으로 처리하는 영역. 알고리즘 검열은 교실 분위기를 망친다.
4. **로그인 / 학생 식별 정보 수집**: 이름은 작품에만 들어간다. 별도 계정 / 학번 / 사진 일절 받지 않음.

## YAGNI로 잘라낸 것 (참고)
- 다국어 (UI는 한국어만)
- 다크모드 (교실 TV는 밝은 환경. 후속에 필요시 추가)
- 작품 검색·필터링 (갤러리 크기 = 한 학기 한 학생 ≤ 30, 스크롤로 충분)
- 협업·실시간 (의도적으로 솔로 활동)

## 성공 기준 (Stage 5 통과 조건)
- 머리글자 "사랑해"를 입력하면 3줄 입력 칸이 정확히 생성된다.
- 작품 저장 후 새로고침 → 갤러리에 그대로 남아있다.
- 갤러리 비어있을 때 룰렛 누르면 "작품이 없어요" 안내. (no crash)
- PDF 다운로드 시 표지·차례·작품·발문 4섹션이 모두 들어있다.
- 모든 버튼이 Tab 키로 도달 가능, focus 윤곽 가시.
